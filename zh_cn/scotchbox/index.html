附录: Scotch Box¶

下面介绍如何将优秀的 Scotch Box 安装到自己的电脑上，以便在几分钟内通过简単的命令就能拥有一个完全可操作的 XenForo 开发环境。

XenForo 有自定义的 Scotch Box 配置，它提供了运行 XenForo 所需的一切，包括调试器和性能增强的数据缓存。

Scotch Box 的运行环境在 VirtualBox / Vagrant 中。

安装 Scotch Box 虚拟机¶

首先确定您希望虚拟 Web 服务器在电脑上保存其文件的位置。建议您在自己的用户主目录中选择一个位置。

在以下示例位置，我们将使用名为 MyServer 的目录，此目录位于您自己用户目录的根目录中，并由您的名称 {username} 标识：

  • /Users/{username}/MyServer （Mac）
  • C:\Users\{username}\MyServer （Windows）
  • /home/{username}/MyServer （某些 Linux 发行版本）
  • /users/{username}/MyServer （其他 Linux 发行版本）

选择位置后，请按照以下步骤操作：

  1. 在电脑上安装 VirtualBox
  2. 在电脑上安装 Vagrant

  3. 使用 git 客户端，clone https://github.com/scotch-io/scotch-box 到 MyServer 目录中。在上面的 Mac 示例位置中使用命令行 Client 端，命令是：

    git clone https://github.com/scotch-io/scotch-box /Users/{username}/MyServer

  4. clone 过程完成后，下载此自定义 Vagrantfile 并复盖在 */Users/{username}/MyServer/Vagrantfile 中创建的 Vagrantfile : 下载自定义 Vagrantfile。

  5. 自定义 Vagrantfile 放置到位后，运行以下命令：

    cd /Users/{username}/MyServer vagrant up

您的 Scotch Box 虚拟机现在已经创建并可以使用。

Note

Scotch Box 还提供了一个 'Scotch Box Pro' 版本的虚拟机，购买价格合理。如果你比较偏向喜欢使用 Scotch Box Pro，请参阅下面的部分 介绍 Scotch Box 和 Scotch Box Pro 在配置和运行之间的差异。

文件都在哪儿？¶

一旦您的 Scotch Box 启动并运行后，您可以将 XenForo 的 PHP 和 JS 文件保留在您的主机上，从而允许您使用您所选择的文本编辑器或 IDE，而虚拟机则负责通过其 Web 服务器编译和提供这些文件。

您将能够通过以下地址在 Web 浏览器中访问新的 Web 服务器：

http://192.168.33.10

Web 服务器将从以下位置提取要提供的文件

/Users/{username}/MyServer/public

如果你希望 XenForo 安装在 http://192.168.33.10/xenforo，你应该把 XenForo 包中 upload 文件夹的内容放到 /Users/{username}/MyServer/public/xenforo 中。

停止和重新启动服务器¶

您可以在任何时候通过运行以下命令停止 Scotch Box 服务器

cd /Users/{username}/MyServer
vagrant halt

... ，并且您可以通过运行重新启动它

cd /Users/{username}/MyServer
vagrant up

Note

虽然 Vagrant / Scotch Box 会在您重新启动电脑时自动关闭，但它不会自动再次启动。

每当您重新启动时，您需要再次运行 vagrant up 命令才能使用服务器。

本指南摘自 Scotch Box 官方文档，该文档位于 https://box.scotch.io

Scotch Box Pro¶

基本的 Scotch Box 需要一些额外的配置（通过自定义的 Vagrantfile 来传递）才能运行 XenForo 2，而 Scotch Box Pro 不需要额外的配置，无需下载额外的包就可以运行 XenForo 2。

要运行 Scotch Box Pro，请从 Scotch Box Pro 网站购买它，在您购买后将会収到一部分运行 git clone 命令的指示说明。

现在您可以使用与上述相同的说明进行安装，唯一的例外是您应该下载 此自定义的 Vagrantfile，而不是 Scotch Box 说明中列出的那个内容。
